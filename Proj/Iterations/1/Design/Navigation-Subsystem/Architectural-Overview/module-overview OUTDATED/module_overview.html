LLHAL: Low-Level Hardware Abstraction Layer

This layer of software is the only direct interface that the program has with the BSP (Board Support Package) and so it is meant to facilitate development in the layers that go above it and make it easier to port the code into a physical platform. It should thus be different from platform to platform as much as is needed to satisfy the requirements of the HLHAL
